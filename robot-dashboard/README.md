# ROS2 LogiFlow

ROS2 기반 물류 패키지 운반 및 분류 자율주행 로봇 프로젝트

## 프로젝트 개요
myAGV 2023 기반 로봇이 패키지를 인식하고 목적지로 자율주행하여 운반 및 분류하는 시스템

## 개발 환경
- Robot: myAGV 2023
- Framework: ROS2
- AI: YOLO (객체 인식)
- Language: Python

## 주요 기능
- 자율주행 (Navigation)
- 패키지 인식 (YOLO)
- 물류 분류
- 시스템 모니터링
